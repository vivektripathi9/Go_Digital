# Components Folder

This folder contains all media assets (images and videos) for the GO-DIGITAL website.

## Folder Structure

```
components/
├── images/          # All image files (JPG, PNG, SVG, etc.)
└── videos/          # All video files (MP4, WebM, etc.)
```

## Usage

### Images
Place all image files in the `images/` folder. You can organize them further by creating subfolders if needed:
- `images/hero/` - Hero section images
- `images/about/` - About section images
- `images/projects/` - Project images
- etc.

### Videos
Place all video files in the `videos/` folder. Current video usage:
- `videos/hero-video.mp4` - Background video for hero section

## File Naming Convention

- Use lowercase letters
- Separate words with hyphens (e.g., `hero-video.mp4`)
- Be descriptive (e.g., `founder-portrait.jpg`)

## Supported Formats

### Images
- JPG/JPEG
- PNG
- SVG
- WebP (recommended for better performance)

### Videos
- MP4 (H.264 codec - recommended)
- WebM (for better browser compatibility)

## Tips

- Optimize images before uploading (compress for web)
- Keep video file sizes reasonable (under 10MB for hero videos)
- Use descriptive filenames for easy reference
