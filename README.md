# GO-DIGITAL Website

A modern, animated website for GO-DIGITAL ENTERPRISES PVT LTD with a black and white theme.

## Features

- **Black & White Theme**: Clean, professional design
- **Smooth Animations**: Scroll-triggered animations and transitions
- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean layout with hover effects and transitions

## Structure

The website includes the following sections:

1. **Hero Section** - Main landing area with company name
2. **About Us** - Company information and expertise
3. **About Founder** - KARAMALA PRADEEP's profile and achievements
4. **What Have We Sold** - Projects and impact created
5. **Our Strategy** - Six strategic pillars
6. **Company Credentials** - Legal and registration details
7. **Contact** - Contact information and thank you message

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - JavaScript for interactivity and scroll animations

## How to Use

1. Open `index.html` in a web browser
2. The website will automatically load with all animations

## Customization

You can edit each section individually:
- Modify content in `index.html`
- Adjust styling in `styles.css`
- Change animation behavior in `script.js`

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
