# Driven Section

This folder contains images and assets for the "Driven" section.

## Section Description

The Driven section appears right after the Hero/Home section and features:
- **Left Side**: "Real Estate" headline and tagline "As the world shifts, build a stronger foundation"
- **Right Side**: Cityscape illustration/image showing construction and development

## Folder Structure

```
driven/
├── README.md (this file)
└── [images/] (add your images here)
```

## Image Requirements

- **Cityscape Illustration**: Image showing buildings, cranes, and construction elements
- Recommended size: 1200x800px or larger
- Format: JPG, PNG, SVG, or WebP
- Aspect ratio: 3:2 or 4:3
- Style: Modern, clean illustration or photograph of cityscape/construction

## Usage

After adding images to this folder, update the HTML file:
- Replace `driven-image-placeholder` div with `<img>` tag
- Use relative paths: `driven/your-image-name.jpg`

## Naming Convention

- Use descriptive names: `cityscape-illustration.jpg`, `construction-cityscape.png`
- Use lowercase with hyphens
- Include file extension
