# Our Approach Section

This folder contains images and assets for the "Our Approach" section.

## Folder Structure

```
approach/
├── README.md (this file)
└── [images/] (add your images here)
```

## Image Requirements

### Part 1: Client Is The King
- **Consultation Image**: Image showing business consultation/meeting
- Recommended size: 1200x800px or larger
- Format: JPG, PNG, or WebP
- Aspect ratio: 3:2 or 4:3

### Part 2: Buildings Become Brands
- **Luxury Property Image**: Image showing luxury property/pool/cityscape
- Recommended size: 1200x800px or larger
- Format: JPG, PNG, or WebP
- Aspect ratio: 3:2 or 4:3

## Usage

After adding images to this folder, update the HTML file:
- Replace `approach-image-placeholder` divs with `<img>` tags
- Use relative paths: `approach/your-image-name.jpg`

## Naming Convention

- Use descriptive names: `consultation-meeting.jpg`, `luxury-property-pool.jpg`
- Use lowercase with hyphens
- Include file extension
